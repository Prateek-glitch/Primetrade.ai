# Binance Futures Testnet Trading Bot

A simplified Python command-line application that places orders on the Binance Futures Testnet (USDT-M). Built with a modular architecture separating API client logic from the CLI layer, ensuring scalability, robust error handling, and comprehensive logging.

## Prerequisites
* Python 3.8+
* A Binance Futures Testnet Account
